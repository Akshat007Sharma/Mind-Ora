# 🧠 Mindora

> *A space to say what you're actually feeling.*

Mindora is an anonymous AI emotional wellness companion built for university students.

Most students won't go to a counselor. They won't always open up to friends either. There's this gap where people just sit with whatever they're feeling because talking about it feels like too big a step. Mindora exists to close that gap.

No login. No judgment. Your conversation memory can be cleared anytime you want.

---

## ✨ What it does

- 💬 Chat anonymously with an AI companion trained to listen and respond with empathy
- 🔒 No accounts, no login. Conversation memory can be cleared anytime
- ⚡ Runs entirely in the browser, no install needed
- 🧩 Single HTML file, no backend, no database

---

## 🛠️ Built With

| Tech | Purpose |

## 💡 Why I built this

University mental health support is either inaccessible, stigmatized, or just not where students actually are. I wanted to build something that met them where they are, with zero friction and complete anonymity.

---

## 🚀 Running Locally

1. Clone the repo
2. Add your Groq API key in the config section of `index.html`
3. Open `index.html` in your browser
4. That's it
